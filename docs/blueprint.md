# **App Name**: Pixel Emporium

## Core Features:

- Home: Página inicial do site de e-commerce.
- Catálogo de Produtos: Navegue e pesquise por produtos gamers (consoles, jogos, acessórios).
- Filtros de Produtos: Filtre produtos por categoria, preço, marca e avaliação.
- Busca de Produtos: Procure por produtos usando uma busca por palavra-chave.
- Paginação de Produtos: Navegue pelas listagens de produtos usando a paginação.
- Detalhes do Produto: Visualize informações detalhadas sobre cada produto, incluindo nome, preço, fotos, descrição, categoria e estoque disponível.
- Adicionar ao Carrinho: Adicione produtos ao carrinho de compras.
- Sobre Nós: Informações sobre a loja.
- Login: Página de autenticação do usuário com campos de e-mail e senha, validação, mensagens de erro, links para registro e recuperação de senha.
- Registro de Usuário: Página de registro de usuário com campos para nome completo, e-mail (único), senha e confirmação, número de telefone e endereço completo. Salva os dados no banco de dados Neon.
- Perfil do Usuário: Área exclusiva com acesso restrito, contendo dados pessoais do usuário, função de edição de dados, histórico de pedidos, carrinho completo, lista de produtos favoritos (opcional) e botão de logout.
- Carrinho de Compras: Funcionalidade para exibir produtos adicionados, alterar quantidade, remover itens e exibir subtotal e total. Inclui um botão de checkout para usuários logados e um prompt de login para usuários não logados.
- Checkout - Revisão de Itens: Passo 1 do processo de checkout: listagem de produtos no carrinho, preço por item, quantidade e total parcial.
- Checkout - Endereço e Dados do Usuário: Passo 2 do processo de checkout: exibe o endereço salvo e permite editar ou adicionar outro endereço. Valida os dados.
- Checkout - Método de Pagamento: Passo 3 do processo de checkout: seleção de métodos de pagamento (sem integração real), campos para inserir as informações necessárias e validações básicas.
- Checkout - Resumo Final: Passo 4 do processo de checkout: exibe o nome do usuário, endereço de entrega, itens do pedido e total final. Inclui um botão para confirmar o pedido.
- Confirmação de Pedido: Gera um número de pedido, salva o pedido e os itens no banco de dados Neon, exibe a mensagem 'Pedido realizado com sucesso' e redireciona para o histórico de pedidos.

## Style Guidelines:

- Cor primária: Roxo Elétrico (#BE29EC) para uma sensação vibrante e emocionante.
- Cor de fundo: Cinza Escuro (#26232D) para criar contraste e destacar os elementos vibrantes.
- Cor de destaque: Fuchsia (#E403A8) para elementos interativos e chamadas para ação.
- Fonte do título: 'Space Grotesk', sans-serif, para uma sensação moderna e tecnológica.
- Fonte do corpo: 'Inter', sans-serif, para descrições claras e legíveis.
- Use ícones personalizados representando vários produtos e categorias de jogos. Garanta que eles sejam visualmente consistentes com o tema geral.
- Layout limpo e moderno com foco nas imagens dos produtos. Use um sistema de grade para exibir os produtos de forma eficaz.
- Animações e transições sutis para melhorar a experiência do usuário (por exemplo, efeitos de foco do produto, atualizações do carrinho).
